End-to-end Loan Prediction project using Python, Google Colab, Machine Learning, Streamlit, and Power BI for data cleaning, analysis, prediction, and interactive visualization.
# Loan Prediction Project

## 📌 Project Overview

This project focuses on analyzing and predicting loan approval outcomes using data analysis, machine learning, and interactive visualization tools.

The project includes data cleaning and preprocessing using Python and Google Colab, exploratory data analysis, a machine learning-based loan prediction application built with Streamlit, and an interactive Power BI dashboard for analyzing applicant and loan characteristics.

## 🛠️ Tools & Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit
- Power BI
- GitHub

## 📂 Project Components

### 1. Data Cleaning & Preprocessing
The dataset was cleaned and prepared using Python in Google Colab. The process included checking the dataset structure, identifying categorical variables, and preparing the data for analysis and machine learning.

### 2. Machine Learning
A machine learning model was developed to predict loan approval outcomes based on applicant information such as:

- Age
- Income
- Loan Amount
- Credit Score
- Years of Experience
- Gender
- Education
- City
- Employment Type

### 3. Streamlit Application
A Streamlit web application was created to provide an interactive interface for loan prediction. Users can enter applicant details and use the trained model to generate a loan approval prediction.

### 4. Power BI Dashboard
An interactive Power BI dashboard was developed to analyze loan applications and applicant profiles.

The dashboard includes:
- Loan approval overview
- Applicant and financial profile analysis
- Loan approval insights
- Key findings and recommendations

## 📊 Key Analysis Areas

The project explores relationships between loan approval and factors such as:

- Age
- Income
- Loan Amount
- Credit Score
- Education
- Gender
- City
- Employment Type
- Years of Experience

## 💡 Insights & Recommendations

The analysis highlights the importance of credit score, income, loan amount, and employment characteristics when evaluating loan applications. Data-driven analysis can help financial institutions improve risk assessment, understand applicant profiles, and support more consistent loan approval decisions.
